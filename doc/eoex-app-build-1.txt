Build a responsive, accessible Netflix-style landing page using HTML, CSS, and vanilla JavaScript (no build step required). Emulate the visual style, layout, and interactions of netflix.com  while avoiding any trademarked logos or proprietary fonts. Include semantic markup and WCAG AA contrast.

Requirements:

Layout:

Hero section with dark background, subtle gradient, centered headline, subhead, and an email signup row with input + primary CTA.

A horizontal “Trending Now” carousel with poster cards and left/right chevrons, snapping scroll on desktop, swipe on mobile.

A “More reasons to join” features grid (4 cards) with icon, title, and short copy.

An FAQ accordion list (expand/collapse with smooth transitions, only one open at a time).

A footer with multi-column links, language selector, legal notices, and small-print copy.

Visuals:

Use the brand kit colors and type below (substitute open-source fonts). Strong emphasis on dark UI, bold red CTAs, clean typographic scale.

Poster cards: aspect ratio ~2:3, hover reveals subtle lift/shadow, focus ring visible for keyboard nav.

CTA buttons: high-contrast red background, white text, rounded-rectangle shape, prominent hover/focus states.

Accessibility:

Semantic elements (header, main, section, nav, footer, h1–h3, ul/li, button).

Keyboard navigable carousels (tab focusable cards, arrow-key support, visible focus).

aria-expanded and aria-controls on accordion buttons; proper labels for inputs.

Behavior:

Email field: basic format validation and inline error messaging.

Carousel: click chevrons scroll by one “page”; drag/swipe on touch; snapping behavior.

Accordion: animated height transition; toggle icon state; collapses siblings when one opens.

Responsiveness:

Mobile-first with breakpoints at 600px, 900px, and 1200px.

Grid to flex as needed; carousels convert to scrollable rows on small screens.

Assets:

Use placeholder posters (solid color blocks or royalty-free images).

Replace any Netflix logos with a generic wordmark (text-only).

Deliverables:

index.html  with semantic sections (hero, carousel, features, faq, footer).

styles.css  with variables for colors/spacing/typography; uses CSS Grid/Flexbox; prefers prefers-reduced-motion.

app.js  for carousel, accordion, email validation.

A11y: ensure focus styles, aria attributes, and keyboard controls.

Implementation details:

CSS: define variables for colors and spacing; use container widths max 1200px; utility classes for spacing; clamp() for fluid type.

Carousel: overflow hidden container, inner track with cards; JS calculates card width and scroll positions; debounce resize.

Accordion: CSS transition on max-height; JS measures content height; add/remove “is-open” class.

Form: email regex for simple check; show error message under input; disable submit until valid.

No external build tools; keep it simple and dependency-free.

Brand kit emulating Netflix’s visual style
Colors
Primary red: #E50914

Background black: #000000

Very dark gray (page background): #141414

White (text on red/dark): #FFFFFF

Mid gray (secondary text): #B3B3B3

Accent dark (panels/cards): #221F1F

Tip: Use high contrast on text (white on black/dark). Reserve red for primary CTAs, focus accents, and highlights.

Typography
Primary font (substitute): Inter, “Helvetica Neue”, Arial, sans-serif

Headline scale:

H1: clamp(2rem, 2.5vw + 1rem, 3.5rem), 700

H2: clamp(1.5rem, 2vw + 0.5rem, 2.5rem), 700

H3: clamp(1.25rem, 1.5vw + 0.5rem, 1.75rem), 600

Body: 1rem (16px), 400–500, tight leading on hero, relaxed leading on body (1.5)

Letter-spacing: Slightly tightened for headlines; normal for body

Fallback guidance: If Inter isn’t available, use system UI sans or “Helvetica Neue”

Spacing and layout
Container max-width: 1200px; side padding: clamp(1rem, 4vw, 3rem)

Vertical rhythm: 24/32/48 spacing steps; larger spacing in hero and between sections

Card grid: 12-column on desktop; stack to 2–4 columns on tablet; single column on mobile

Components
Hero: Large bold headline, supportive subcopy, email input + “Get Started” button in a single row on desktop, stacked on mobile.

CTA button: Red background (#E50914), white text, 600 weight, 12px–16px radius, box-shadow on hover, visible outline on focus (#FFFFFF 2px).

Carousel card: 2:3 aspect, subtle gradient overlay, hover lift (transform: translateY(-2px)), focus ring.

Accordion: Dark panel background (#221F1F), white text; chevron rotates on open; smooth height transition.

Footer: Multi-column link lists, muted gray text, language selector styled as minimal select; legal small-print in lighter gray.

// Email validation
const form = document.querySelector('.signup');
const emailInput = document.querySelector('#email');
const errorEl = document.querySelector('.signup .error');

form.addEventListener('submit', (e) => {
  e.preventDefault();
  const email = emailInput.value.trim();
  const isValid = /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
  if (!isValid) {
    errorEl.textContent = 'Please enter a valid email address.';
    emailInput.setAttribute('aria-invalid', 'true');
    emailInput.focus();
  } else {
    errorEl.textContent = '';
    emailInput.removeAttribute('aria-invalid');
    alert('Thank you!'); // replace with real handling
  }
});

// Carousel
const track = document.querySelector('.track');
const prevBtn = document.querySelector('.prev');
const nextBtn = document.querySelector('.next');

let cardWidth = 0;
function updateCardWidth() {
  const firstCard = track.querySelector('.card');
  if (firstCard) {
    cardWidth = firstCard.getBoundingClientRect().width + parseFloat(getComputedStyle(track).gap);
  }
}
function scrollByCards(dir = 1) {
  track.parentElement.scrollBy({ left: dir * cardWidth * 3, behavior: 'smooth' });
}
window.addEventListener('resize', () => { updateCardWidth(); });
updateCardWidth();

prevBtn.addEventListener('click', () => scrollByCards(-1));
nextBtn.addEventListener('click', () => scrollByCards(1));

// Accordion
document.querySelectorAll('.accordion .item').forEach((item) => {
  const btn = item.querySelector('.trigger');
  const panel = item.querySelector('.panel');

  btn.addEventListener('click', () => {
    const open = item.classList.toggle('is-open');
    btn.setAttribute('aria-expanded', open ? 'true' : 'false');

    // Close others
    document.querySelectorAll('.accordion .item').forEach((other) => {
      if (other !== item) {
        other.classList.remove('is-open');
        other.querySelector('.trigger').setAttribute('aria-expanded', 'false');
      }
    });
  });
});
